# 快速启动指南

## 首次运行

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动应用**
   ```bash
   python manager.py
   ```

3. **访问应用**
   打开浏览器访问: http://127.0.0.1:5000/

## 常见问题

### Q: 如何安装依赖？
A: 运行 `pip install -r requirements.txt`

### Q: 如何运行？
A: 运行 `python manager.py`（注意是 manager.py 不是 mangager.py）

### Q: 支持哪些图像格式？
A: PNG, JPG, JPEG, GIF, BMP, TIFF, WEBP

### Q: 最大文件大小是多少？
A: 50MB

### Q: 处理后的图片在哪里？
A: 在 `assets/imgs/final/` 目录

## 停止应用

在终端按 `Ctrl + C`
